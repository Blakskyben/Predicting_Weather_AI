About the Project

Ben came up with this project a few months ago, the date being 13/6/19 as of writing this. So what exactly is this project aiming to do? This project, which Ben calls "Predicting Weather with AI", is exactly what the title says. The program takes some weather conditions and "spits" out what will happen next. An example of this is, let's say Ben puts a pressure of 1019 and a humidity of 23. Then the program would output, or show you what would happen, like it would be clear skies that day. As Ben stated above, he wants to use his skills to help the world. When he came up with this idea, he realized, that if he could create this, it could actually help to warn people about upcoming weather conditions. Ben hopes to be able to spread this program throught the internet, and possibly farther to help people.
